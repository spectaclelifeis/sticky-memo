# 📌 Sticky Memo App
파이썬 `tkinter`로 제작된 가볍고 직관적인 스티커 메모 애플리케이션입니다.

## 📥 다운로드 및 실행 방법
1. 오른쪽 **Releases** 섹션에서 최신 버전의 `StickyMemo.exe` 파일을 다운로드하세요.
2. `StickyMemo.exe`를 실행하면 바로 사용할 수 있습니다.

## ✨ 주요 기능
- **자동 저장**: 메모를 입력하거나 수정하는 즉시 `memos.json` 파일에 자동으로 저장됩니다.
- **데이터 관리**: 메모 전체 데이터를 JSON 형식으로 내보내거나(`e`) 다시 불러올 수(`i`) 있습니다.
- **다양한 정렬**: 가나다순, 최신순, 오래된순, 수정순으로 메모 리스트를 자유롭게 정렬할 수 있습니다.

## ⌨️ 단축키 안내
프로그램을 더 빠르게 사용하려면 아래 단축키를 활용해 보세요.

### 메인 리스트 및 공통
- `Ctrl + N`: 새 메모 만들기
- `Ctrl + F`: 검색창으로 포커스 이동
- `Enter`: 선택한 메모 열기
- `Delete`: 선택한 메모 삭제
- `?`: 전체 단축키 도움말 보기

### 데이터 및 정렬
- `e`: 메모 내보내기 (Export)
- `i`: 메모 불러오기 (Import)
- `a`: 가나다순 정렬
- `s`: 최신순 정렬
- `d`: 오래된순 정렬
- `f`: 수정순 정렬

### 메모 팝업 창
- `Esc`: 현재 메모 창 닫기 (자동 저장됨)
